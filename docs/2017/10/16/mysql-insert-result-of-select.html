MySQL: Insert Result of Select query

16 Oct 2017
  • mysql

You don’t need to use VALUES after INSERT INTO. Just use SELECT after INSERT INTO.

INSERT INTO
  table_name(col1, col2, ...)
SELECT
  col1, col2, ...
  FROM another_table_name;
